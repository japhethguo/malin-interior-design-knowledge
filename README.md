# 玛琳的室内设计知识项目

这是一个用来沉淀、整理、训练和交付室内设计经验的长期项目。

项目的第一位作者是玛琳：一位真实做过许多空间、客户、预算、材料、审美与落地决策的室内设计师。这里要保存的不是“漂亮话”，而是她多年工作里真正可复用的判断、方法、案例、经验、失误、直觉和审美标准。

项目的长期目标，是把这些经验转化成可以被人类和 AI 学习、调用、传播与交付的知识系统，帮助更多人成为更好的室内设计师，也帮助普通人在装修、布置、选材、软装和日常居住中做出更好的决定。

## 核心问题

这个项目围绕一个问题展开：

> 玛琳到底是如何判断一个空间应该怎么被设计、被使用、被实现、被感受的？

围绕这个问题，我们会持续收集并回答：

- 她如何理解客户、家庭、生活方式与真实需求？
- 她如何看户型、动线、采光、收纳、尺度和功能冲突？
- 她如何处理预算、材料、施工、审美、耐用性之间的取舍？
- 她如何形成一个空间的风格、气质和情绪？
- 她如何判断一个方案“还不够好”或者“已经成立”？
- 她踩过哪些坑，又形成了哪些可提前规避的原则？

## 项目结构

- `00_项目罗盘/`：项目愿景、原则、对象、边界和阶段规划。
- `01_原始素材/`：访谈、随笔、语音转写、聊天记录、灵感片段和未整理想法。
- `02_案例库/`：真实项目、虚拟案例、空间改造、客户问题和设计决策复盘。
- `03_方法论/`：从经验中提炼出的判断框架、流程、清单和设计原则。
- `04_教学与交付/`：课程、文章、工作坊、咨询包、练习题和评估标准。
- `05_AI传输包/`：给 AI 使用的背景设定、提示词、知识卡片、问答库和评审标准。
- `06_运营与产品化/`：受众、产品形态、发布节奏、品牌表达和商业实验。
- `99_模板/`：可重复使用的记录模板。

## 使用方式

1. 先把玛琳的真实经验放进来，不急着整理得漂亮。
2. 每隔一段时间，把原始素材提炼成案例、原则、清单和训练题。
3. 对每条经验追问：它能不能帮助别人做出更好的设计判断？
4. 对每个方法追问：它能不能被另一个人或另一个 AI 稳定复现？
5. 最后再把成熟内容打包成课程、指南、咨询工具或 AI 助手能力。
